# 📊 Global Superstore Business Intelligence Dashboard

An interactive Business Intelligence dashboard built in **Microsoft Power BI** using the Global Superstore dataset. The dashboard provides insights into sales performance, profitability, customer behavior, product categories, and geographical trends through a clean and interactive interface.

---

## 🚀 Dashboard Overview

This project contains **3 interactive dashboard pages**:

### 📈 Executive Overview
- Total Sales
- Total Profit
- Total Orders
- Repeat Customer Rate
- Monthly Sales Trend
- Top Regions by Sales
- Profit by Category
- Top Countries by Sales
- Orders by Ship Mode

---

### 📦 Products & Categories
- Top Performing Category
- Worst Performing Sub-Category
- Average Discount
- Unique Products
- Sales by Sub-Category
- Profit Waterfall Analysis
- Top 5 Products by Revenue
- Profit vs Discount Analysis

---

### 🌍 Geography & Customers
- Total Markets
- Unique Customers
- Top Market
- Average Order Value
- Sales by Country
- Sales by Market
- Profit by Customer Segment
